!DOCTYPE html: Selects most current version of html; required for valid HTML website

attributes: none

Example:<!DOCTYPE html>

html: HTML tag, marks the beginning and end of your HTML code; required for valid HTML website

attributes: none

Example: <html>Website</html>

head: Head tag, conatins things for your web browser to think about; required for valid HTML website

attributes: none

Example: <head>Hi</head>

a: Anchor tag, makes text into a hyperlink

attributes: href

Example: <p><a href="https://www.youtube.com/">link</a></p>

body: Body tag, contains content of your website(pictures, text, videos, etc.); required for valid HTML website

attributes: none

Example: <body>paragraph</body>

h1: Header tag, makes text large and puts on separate line 
 
attributes: none 

example:<h1>Mr. Pelletier</h1>

h6: Reader tag, like h1 but with smaller text. There are header tags from h1 to h6, with h6 being the smallest. Good for subheadings.

attributes: none

Example: <h6>Hi</h6>

P: Paragraph tag 

attributes: none

Example: <p>Hi. My name is Name.</p>

ul: Unordered list, sorts text into a bulleted lists

attributes: none

Example: <ul><li>one</li><li>two</li><li>three</li></ul>

ol: Ordered list, sorts text into a numbered list

attributes: none

Example: <ol><li>one</li><li>two</li><li>three</li></ol>

li: List tag, used to specify list item in unordered, ordered, directory, and menu lists.

attributes: none

Example: <ol><li>one</li><li>two</li></ol>

img: Image tag, inserts an image file

attributes: src

Example: <img src="towns_44084.jpg">

title: Title tag, puts a name on your website; required for valid HTML website

attributes: None

Example<title>Title</title>
